# Sales Data Analysis

## Objective
This project analyzes sales data to identify revenue trends across different cities and products. The goal is to derive actionable insights that support business decisions.

## Tools Used
- Python
- Pandas
- Matplotlib

## Description
- Cleaned and processed the sales dataset, handling missing values and transforming columns as needed.
- Created a derived `Revenue` column to calculate net sales value.
- Performed city-wise and product-wise revenue analysis using groupby and aggregation.
- Identified top-performing products and top revenue-generating cities.
- Visualized key insights using bar, line, and pie charts.

## Insights
- Mumbai generates the highest total revenue among all cities, indicating stronger sales performance in that region.
- Laptops contribute the most revenue among products, suggesting strong demand.
- Trends indicate opportunities for improving sales performance in less dominant regions or products.

## Files Included
- `sales.ipynb`: Jupyter notebook with full code and analysis
- `cleaned_sales_data.csv`: The cleaned dataset used in the project
